Parses a bitmap from an input stream. 

It's length and width will be read as the first two tokens, and followed by a number of 0s and 1s corresponding respectively to unset and set bits.

\begin{DoxyRemark}{Remarks}
A length or width of 0 stops the parsing.
\end{DoxyRemark}
